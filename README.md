# 📚 About This Repository

This repository contains all source codes, examples, exercises, algorithms, and projects used in the **Darsman Python Programming Course**.

The course is designed for students who want to learn Python from scratch and continue to an intermediate level using practical and project-based learning.

---

# 🎯 Course Goals

By the end of this course, students will be able to:

- Understand Python fundamentals
- Write clean and readable code
- Work with data structures
- Build command-line applications
- Understand Object-Oriented Programming (OOP)
- Work with files and modules
- Implement basic algorithms
- Use popular Python libraries
- Build complete Python projects


